# GoIPLookUP
A tool to lookup IP address from URL

### Usage of GoIPLookUp:
```
-v, --verbose   [ verbose output { default: false } ]
-h, --help      [ prints help information ]
-u, --url       [ website url ]
-p, --path      [ file path for bulk urls ]
-o, --output    [ output filename ]
```
